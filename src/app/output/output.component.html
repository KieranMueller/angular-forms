in this example

  • we create a parent component with a variable 'messageFromChild' that we
    expect to receive from the child component
  • we include the value of this variable in our template, initially it is an empty string
  • we include the child component selector in the parent's template
  • we create a method in the parent component, to handle the value emitted from
    the child, setting it equal to our 'messageFromChild' variable
  • in the child, we create a variable 'message' that will hold the value we'd like to
    emit to the parent - using two way binding with an input and [(ng-model)]
  • we create the event emitter named 'emitter'
  • we create a button with a click event, that fires the 'emitEvent()' function
  • in this function, we emit our EventEmitter 'emitter', passing
    the value of 'message' to the parent
  • now, we must receive the event in the parent template, and call our
    'receiveEvent()' function, passing the value emitted from the child as '$event'
