Why use local storage?

Local storage can make your application quicker, and more efficient. Instead of
server side storage, fetching data from a database, we can make use of
client side storage and store data in the client's web browser. It can allow you
to cut down on unnecessary HTTP requests, possibly storing a large amount
of data from an initial HTTP GET request when a user enters the site, and
fetching/using the data stored in local storage for the duration of the user's visit.
Be cautious storing sensitive information, as the key value pairs saved in
local storage are easily accessible through the developer console on your machine.

After entering data and hitting the save button...

  • Open the developer console (Ctrl + Shift + i)
  • In the top bar, navigate to 'Application'
  • On the left hand side, within the 'Storage' section, expand 'Local Storage'
  • View the key value pairs that you just stored
  • Click the 'Retrieving data' section in the navigation bar to learn how to retrieve this data!
