Sharing data between unrelated components

  • here we are using a shared service called 'BehaviorService'
    • we create a behavior subject
    • we create an observable from the behavior subject
    • within our three identical components we
      • inject the service
      • on init, we subscribe to this observable, setting
        our local message variable to the current value
      • we create a method named updateMessage that
        calls the updateMessage method in the service
