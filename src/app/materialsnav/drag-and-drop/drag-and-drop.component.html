Drag me anywhere!

Drag an html element anywhere on the page

  • Add this line in the NgModule where you'd like to use this feature (typically your app.module.ts file)
    • import {{ "{" }} CdkDrag {{ "}" }} from '@angular/cdk/drag-drop';
  • Add this line to the imports array
    • CdkDrag
  • Add this directive to any element you'd like to drag!
    • cdkDrag
  • And you're done! Much easier than writing a ton of javascript yourself
