What are signals?

  • Taken straight from the Angular docs: A signal is a wrapper around a value that can notify interested consumers when that value changes. Signals can contain any value, from simple primitives to complex data structures.
  • Notice that signal values are read from a getter function created behind the scenes
    • This is why we are interpolating our variable as num() with parenthenses in the template, it is returning the result of the getter function
  • We can use the set method to change the value
  • We can use the update method to update the value based on its previous value
