in this example

  • the parent declares a variable 'count' (but does not display its value in the template)
  • the parent template holds a button. When clicked, the value of 'count' is incremented
  • the child component's selector is included in the parent's template,
    making it a child of the parent component
  • within the child selector, we use property binding to bind the 'count' variable
    from the parent, to the 'count' variable in the child component
  • within the child component typescript, we create a variable 'count' prefixed
    with Input(), denoting that this value will be passed to us from the parent
  • we display the value of 'count' in the child's template
