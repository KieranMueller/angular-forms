What's going on here?

  • If the buttons are disabled, you first need to visit the 'Saving data'
    section and enter data to store in local storage
  • The bulk of logic in the ngOnInit() function can be ignored, it is
    irrelevent, simply for the sake of enabling or disabling the buttons
  • We have two buttons, one removes your data, one removes your pet's data
  • localStorage.removeItem() takes one argument, the key you'd like to
    delete, if it exists, it removes the keypair entirely
    (both key and value), if it does not exist, well nothing happens!
