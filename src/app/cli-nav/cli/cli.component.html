What is it?

  • The Angular Command Line Interface (CLI) allows you to initialize, develop, scaffold, and maintain Angular applications from a command shell
  • Any Angular developer should be well rounded with Angular CLI

Steps

  • You must have npm installed, which requires Node.js to be installed as well
    • If you are unsure whether or not npm is installed, open a command prompt and run 'npm -v' or 'npm version'
    • If you need to install Node.js and npm, follow this link install
  • Once you have installed npm, copy and run the command above 'npm install -g @angular/cli' to install Angular CLI
  • Once installed, run 'ng v' or 'ng version' to confirm installation and view the current version
