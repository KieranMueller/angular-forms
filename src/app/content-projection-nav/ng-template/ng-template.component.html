ng-template is a placeholder for conditionally rendering HTML

  • Unless ng-template is conditionally rendered, content inside will not be displayed
  • *ngIf cannot be used directly on ng-template tag
    • behind the scenes, ng-template generates an *ngIf,
      so this would violate Angular's 'no more than one structural directive per tag' rule
  • However, [ngIf]="expression" CAN be used on ng-template
  • Notice in this example, we place a tag on the ng-template to render the HTML
    inside of it when the else condition is called within the *ngIf of the first div
